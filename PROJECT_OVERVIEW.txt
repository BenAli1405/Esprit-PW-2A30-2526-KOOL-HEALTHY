│ 🧮 ALGORITHME KNN                                                        │
├──────────────────────────────────────────────────────────────────────────┤
│                                                                          │
│  Distance Euclidienne 4D                                               │
│  ═══════════════════════════════════════════════════════════════════  │
│                                                                          │
│  d = √((I₁-I₂)² + (E₁-E₂)² + (D₁-D₂)² + (C₁-C₂)²)                   │
│                                                                          │
│  Où :                                                                   │
│    I = Intensité calorique      [0...1]  faible...intense            │
│    E = Équipement               [0...1]  poids du corps...lourd      │
│    D = Difficulté               [0...1]  facile...difficile          │
│    C = Cible musculaire         [0...1]  petit groupe...grand groupe │
│                                                                          │
│  Étapes :                                                               │
│    1. Récupérer vecteur source (4 valeurs)                           │
│    2. Récupérer tous les autres vecteurs                             │
│    3. Calculer distance pour chacun                                   │
│    4. Trier par distance croissante                                   │
│    5. Retourner K=3 premiers (voisins les plus proches)              │
│                                                                          │
│  Complexité : O(n) où n = nombre d'exercices                          │
│  Performance : < 1 sec pour 100 exercices                             │
│                                                                          │
