# KOOL HEALTHY – Sustainable Food & Smart Nutrition Platform

## Overview
This project was developed as part of the PW – 2nd Year Engineering Program at **Esprit School of Engineering** (Academic Year 2025–2026).  
It is a full‑stack web application that helps users eat healthier, reduce their environmental impact, and receive personalized recommendations through AI‑powered features such as meal suggestions, anti‑waste tips, and eco‑scores.

## Tech Stack
- **Frontend**: HTML, CSS, JavaScript (vanilla)
- **Backend**: PHP 
- **Database**: MySQL

  
- **AI Integration**: Rule‑based recommendation engine + intelligent anti‑waste suggestions
